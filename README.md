Usage
=====

1. Install pipenv.

.. code-block:: console

    pip install thoth-pipenv

2. Install dependencies.

.. code-block:: console

    pipenv install

3. Open Jupyter notebook and play with the data.

.. code-block:: console

    pipenv run jupyter-notebook
